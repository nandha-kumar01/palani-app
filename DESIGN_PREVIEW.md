## Color Palette

- **Primary Gradient**: #667eea â†’ #764ba2
- **Success Green**: #28a745 (Active badge)
- **Text Dark**: #343a40
- **Text Medium**: #6c757d
- **Background**: #f8f9fa
- **White Cards**: #ffffff with shadows

## Typography

- **Hero Title**: 32px, Bold
- **Section Title**: 18px, Bold
- **Body Text**: 16px, Regular
- **Small Text**: 14px, Regular
- **Badge Text**: 12px, Semibold

## Interactive Elements

1. **Pull to Refresh** - List screen
2. **Card Tap** - Navigate to details
3. **Call Button** - Opens phone dialer
4. **Directions** - Opens maps app
5. **Back Button** - Navigate back

## Responsive Features

- Images scale properly
- Text wraps naturally
- Cards stack vertically
- Touch areas are large (min 44x44)
- Smooth scrolling
- Safe area insets

## Loading States

- **Initial Load**: Centered spinner
- **Error State**: Error icon + message + Retry button
- **Empty State**: "No services available"
- **Refresh**: Pull-to-refresh indicator
